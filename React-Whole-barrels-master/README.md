# React-Whole-barrels

一个完整的前端脚手架，涵盖webpack+es6+react+redux等技术栈；
该demo实现了简单的redux效果，简单易理解，适合新手以及想要自己搭建脚手架的朋友


## 技术栈 ##
### react + react-router + react-redux + es6 + webpack + less + antd  ###



## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8089
npm run dev
```
